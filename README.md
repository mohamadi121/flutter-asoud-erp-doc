# ASOUD ERP Design Checkpoint 01

این بسته، پشتیبان مرحله اول طراحی ASOUD ERP برای بارگذاری در مخزن `mohamadi121/flutter-asoud-erp-doc` است.

## وضعیت مرحله

- جریان تنظیمات پایه و ایجاد دفتر طراحی شده است.
- شش صفحه جدید با پیشوند `NEW-BASE` در Figma ایجاد شده‌اند.
- صفحات قدیمی حذف یا جایگزین نشده‌اند.
- ظاهر مرجع همان PDF تأییدشده است: فونت Vazirmatn، کارت‌های سفید، دکمه‌های آبی و آیکن‌های رنگی.
- Prototype از شروع ایجاد دفتر تا بازبینی نهایی متصل شده است.
- دکمه نهایی «ایجاد دفتر» پس از طراحی داشبورد به داشبورد و پیام موفقیت متصل می‌شود.
