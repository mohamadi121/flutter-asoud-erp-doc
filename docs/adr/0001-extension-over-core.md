# ADR-0001: افزونه به‌جای تغییر هسته ERPNext

## وضعیت

پذیرفته‌شده

## تصمیم

تمام نیازهای اختصاصی حسابداری ایرانی در Frappe App مستقل `asoud_erp` پیاده می‌شوند. تغییر مستقیم هسته ERPNext تنها زمانی مجاز است که Extension Point مناسب وجود نداشته باشد و تصمیم جدید ثبت شود.

## نتیجه

- ارتقای ERPNext ساده‌تر می‌شود.
- اختلاف با upstream کاهش می‌یابد.
- تست و بازگشت نسخه اختصاصی مستقل خواهد بود.
