# جریان Prototype

## اتصال صفحات

- شروع ایجاد دفتر → انتخاب نوع شخصیت و فعالیت
- ادامه → اطلاعات هویتی و مالیاتی
- ادامه → تنظیمات مالی و سال مالی
- ادامه → نقش‌ها و کدینگ پیشنهادی
- ادامه → بازبینی نهایی
- بازگشت در هر مرحله → مرحله قبلی

## پایان جریان

صفحه موفقیت مستقل حذف شده است. پس از انتخاب «ایجاد دفتر»، کاربر وارد داشبورد اصلی می‌شود و پیام موفقیت داخل همان داشبورد نمایش داده می‌شود. جزئیات عملیات و سابقه ایجاد دفتر قابل مشاهده باقی می‌ماند.
