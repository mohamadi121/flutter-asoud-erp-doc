# قوانین ثابت استایل

- PDF تأییدشده مرجع قطعی طراحی است.
- فونت اصلی: Vazirmatn Regular/Bold.
- ابعاد مرجع موبایل: 340 × 844.
- پس‌زمینه روشن و کارت‌های سفید با گوشه گرد.
- دکمه اصلی آبی و دکمه ثانویه سفید با کادر کم‌رنگ.
- آیکن‌های رنگی آبی، بنفش، سبز و نارنجی باید حفظ شوند.
- برای صفحات جدید زبان بصری جدید تعریف نمی‌شود.
- صفحات نمایشی حالت‌های کشویی داخل Design System نگهداری می‌شوند.
- صفحات وب ایجاد دفتر از Prototype موبایل جدا باقی می‌مانند.
- تمام صفحات جدید تا زمان تأیید با پیشوند `NEW` مشخص می‌شوند.
